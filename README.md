This project is a desktop application developed using Java as part of an educational initiative to improve software development skills. The primary goal of this project is to understand the implementation of GUI-based applications and effective database connectivity.

🎯 Educational Purpose
The core objective of this project is to gain practical experience in:

Java Swing/AWT: Building interactive user interfaces.

Database Management: Handling data operations (CRUD - Create, Read, Update, Delete) using SQL/JDBC.

Object-Oriented Programming (OOP): Implementing modular code structures and clean architecture.

Logical Problem Solving: Managing real-world business scenarios through code.

🚀 Key Features
User-Friendly GUI: Intuitive forms for data entry and navigation.

Data Persistence: Seamless connectivity with the database for data storage and retrieval.

Modular Architecture: Organized source code for better maintainability and readability.

🛠 Tech Stack
Language: Java

UI Framework: Java Swing
